# JIRA Operations Reference

Comprehensive reference for JIRA operations including user story management best practices, JQL
query patterns, SCRUM ceremony integration, reporting metrics, and troubleshooting guides.

## When to Use

- Managing JIRA user stories (creating, updating, reviewing, improving)
- Writing or optimizing JQL queries for sprint and backlog management
- Supporting SCRUM ceremonies (planning, daily scrum, review, retrospective)
- Troubleshooting JIRA authentication, permission, or data validation issues
- Reviewing story quality metrics and team performance data
