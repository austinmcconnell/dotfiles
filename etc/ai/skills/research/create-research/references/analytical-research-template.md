## Overview

context for someone unfamiliar to understand the rest of the document.

## Framework / Taxonomy

The organizing structure for the research — classification systems, scales, standards, or categories
that the rest of the document builds on. Include mapping tables if multiple frameworks exist for the
same domain (e.g., CEFR ↔ ILR scale equivalences). Explain where mappings are approximate.

## Summary Table

At-a-glance reference table covering all items in scope. Columns should capture the key dimensions
the reader needs to compare (e.g., category, quantitative estimates, difficulty, notes). Include
methodology notes and caveats below the table when estimates involve extrapolation or significant
uncertainty.
