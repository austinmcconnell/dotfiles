# TrueNAS for a 10″ Mini Rack (DeskPi T0/T1/T1 Plus)
**Date:** 2025-10-31
**Last updated:** 2025-10-31

A concise, source-backed design for a quiet, low-heat TrueNAS/ZFS box that fits a 10″ mini rack. It proposes three architectures (SATA-first Mini‑ITX, embedded SBC with native SATA, and a mini‑PC with multiple M.2), compares 1U vs 2U tradeoffs, gives ZFS layout and power guidance, and lists US‑sourced BOMs. Quick-start below; details in the linked files.
